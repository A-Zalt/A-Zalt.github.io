1.92

Changes from 1.91:

- Removed the "Toggle Copy" button from the editor pause menu
- Object limit was increased from 15000 to 20000
- Added "groups" (layers) and their respective "Edit Group" button to the editor
- Replaced Music/SFX checkboxes with volume sliders
- Changed the advanced settings menu to the layout that's still used today, being under an Options button in the Options instead of a cog button at the top right. Most options now have their own infoboxes. Auto-Retry and Auto-Checkpoints were moved into this menu
- Added Small Editor Step, Smooth Fix and Higher Audio Quality options
- Added 30 new loading messages, as well as the "Loading resources" message
- Fixed bug with saving colors of objects
- The colored singular brick objects no longer have a hitbox (seriously RobTop?)
- Added Steamrolling! and SteamStep icons
- Added UHD quality textures (PC only)
- Added a button to remove all custom songs
- The level browser infobox has now been fixed to correctly reflect the color of custom songs as Purple instead of Orange

Not-in-game content

- Save file binaryVersion changed from 20 to 24
- Manifest version changed from 24 to 25
