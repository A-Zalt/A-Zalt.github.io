1.40

Changes from 1.30:

- Added new main level & song: Clutterfunk
- Added 4 new ships (Clutterfunk!; Demonic Guardian; Epic Collector; Epic Master Collector)
- Added new magenta player color
- Changed appearance of default ship
- Added mini & normal size portal
- Added saws and their respective decorative objects
- Added black tileset
- Changed the appearance of the unlisted tileset from 1.3 and it's no longer unlisted, but the corner block was removed
- Added small spike
- Added Line color trigger
- Added Magic tab to Search
- Ball clicks are no longer buffered after orbs
- Added ball snapping bug
- The New button in the level editor selection menu is now on the bottom
- Your custom levels are now paginated and have a counter on the top right. This is due to MyLevelsLayer being replaced with LevelBrowserLayer
- Your custom levels are also sorted differently
- Object ID 83 now has glow, which was missing before
- The Edit Object button from the Edit Tab was moved to the right side of the editor. It only appears when a color trigger is selected
- Added Facebook link for DJVI in the Download Soundtracks menu
- Links for the soundtracks in the Download Soundtracks menu now go to YouTube instead of Newgrounds
- The ground was very slightly tweaked. It is now a pixel higher and the ground line is no longer tinted by the ground color
- The Facebook button on the main menu now checks for your Internet connection
- Portals (and other large objects) now only take up 1 grid space, allowing you to place them closer to each other
- Ground spikes now render on the same layer as most other objects. This can be seen in the level Dry Out, where some ground spikes appear below regular spikes, which doesn't happen in 1.3
- Fixed a bug where pressing the back button twice from a level you opened in the Featured tab would take you to the Search tab
- Fixed a bug where pressing the back button on a saved level that has an empty level string would take you back to search results (this crashed the game if you hadn't searched anything this session)

Not-in-game content

- Added unlisted Object color trigger
- Manifest version changed from 10 to 11
- Game version changed from 4 to 5
- Save file binaryVersion changed from 4 to 5
- Removed unused 1.3 ship designs from spritesheet
- Added unlisted official level "Active". It had no level data other than song guidelines
- Added RobTop-only Set Featured menu
- Fixed down arrow in RobTop-only Set Stars menu. In 1.3, it worked like the up arrow
