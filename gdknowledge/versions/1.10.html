1.10

Changes from 1.02:

- New main level & song "Time Machine"
- Mirror portals officially added
- Added comments
- Added dislike functionality
- Added 1 new icon for completing Time Machine
- Added white color, one for completing Time Machine in practice and the other for jumping 20000 times
- Icons are now split in 2 rows
- The "More by this user" button in the level info menu has been moved next to the username and shortened to "More" due to the addition of comments
- Pause button now has an outline
- The "Level download failed" popup can no longer appear before the LevelInfoLayer of the level has been fully loaded
- Non-solid black objects (this includes only the decorative grid block in this version) are now layered below most other objects, with the exception of ground spike deco and chains
- Your saved levels are now sorted top to bottom from most recently downloaded to least, as opposed to being sorted the other way around

Not-in-game content

- Manifest version changed from 4 to 5
- Game version changed from 1 to 2
- Your own levels are now stored in CCLocalLevels.dat
- The endpoint likeGJItem.php is now used for liking/disliking levels instead of likeGJLevel.php
- The game spritesheet now has unused textures that only appear in Geometry Dash Lite, and a button that would later become the Report button
- Added unused kEnableTutorial key to CCGameManager, which is set to false by default and does nothing in this version. However, in GD Lite 1.00 and 1.01, it unlocks the editor
