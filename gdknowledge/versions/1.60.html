1.60

Changes from 1.51:

Levels:

- Added 2 new official levels & songs, Electroman Adventures & Clubstep
- Added Secret Coins, their respective statistic and made leaderboards display them. Every official level was retroactively updated to add 3 secret coins each
- Added Map Packs and their respective statistic
- Added infoboxes for official & online levels, which show your attempt count, jump count, normal mode percentage and practice mode percentage
- Added Auto difficulty and added it as a search option

Editor:

- Added many objects, including: invisible objects, fake spikes, jagged spikes, pulsing square, triangle and X, 2 new tilesets, breakable block, a new type of saw, tiny default block and slab and miscellaneous decoration
- Added "Delete Start Pos" to Delete tab in the editor
- Object limit was increased from 6000 to 8000
- Level's object count is now displayed at the top left of the editor pause menu
- Added Tint Ground to BG Color Trigger
- Added portal bounds preview checkbox
- BPM guidelines are now off by default

Achievements/Icons:

- Overhauled icon kit menu. Now there are 4 tabs for each gamemode, and secret coins are displayed
- Achievements menu is now paginated.
- Added new achievements for beating Electroman Adventures in practice & normal mode
- Added new achievements for beating Clubstep in practice & normal mode
- Added new achievements for getting 5, 10, 15, 20, 25, 30, 35, 40, 45 and 50 secret coins
- Added new achievements for beating 1 and 5 Map Packs
- Attempt achievements have been renamed. 100: "U can do it!" -> "Trial and error"; 500: "Trial and error" -> "Crash Tester"; 2000: "Crash Tester" -> "You Shall Not Pass!"

Bugfixes

- Fixed a bug which allowed you to clip through saws when going through a mirror portal
- You can no longer move objects to the left of the starting area
- Fixed an attempt farm bug where you can place a start position inside of a spike and repeatedly hit the restart button to die faster

Miscellaneous

- Theory of Everything soundtrack link no longer goes to http://www.robtopgames.com/theoryofeverything, goes to the Newgrounds upload instead
- Added Top Week leaderboard
- Info box in Creator Leaderboard now mentions that featured levels give 2 CP
- Added Low Detail Mode to options menu
- Gamemode portal textures have been altered. The dots are now larger
- The endscreen now has the editor button for local levels
- Added the "Star rating guide" informational popup when clicking the Rate Stars button for the first time

Not-in-game content

- Manifest version changed from 13 to 14
- Game version changed from 6 to 7
- Save file binaryVersion changed from 6 to 7
- Removed unused Boomlings leftover CCGameStore.dat file
- Partially removed promo system
- Added close button to Set Featured menu
- Added green arrows to Set Featured menu, which change the value by 10
- Default value in Set Featured menu was changed from 20 to 100
- Added RobTop-only Level Replay system. This system was most likely used for the Clubstep v2 video on RobTop's YouTube channel
- Added unused RobTop-only Star Award button
- Removed 109 CCLog calls from 1.51 (180 -> 71)
